# Auto Service Desk

Учебный проект по дисциплине **«Конструирование интернет-приложений»**.  
Приложение представляет собой систему для работы с заявками клиентов автосервиса.

## Что реализовано

### Авторизация и роли
- регистрация и вход пользователей
- JWT-аутентификация
- восстановление сессии
- роли:
  - **Client** — клиент автосервиса
  - **Master** — мастер, работающий с заявками
  - **Admin** — администратор системы

### Административная часть
- управление категориями услуг
- просмотр пользователей
- смена ролей пользователей

### Работа с заявками
- создание клиентом новой заявки
- просмотр списка своих обращений
- просмотр карточки заявки
- фильтрация по статусу и категории

### Очереди мастера
- новые заявки
- назначенные мне
- завершённые заявки
- возможность:
  - взять заявку в работу
  - перевести в статус `InProgress`
  - завершить заявку
  - отклонить заявку

### Тестирование клиентской части
- настройка **Vitest**
- тесты для:
  - login / register / logout / init
  - защиты маршрутов
  - ролевого интерфейса
  - состояний страниц (`loading`, `error`, `empty`)

## Стек технологий

### Backend
- ASP.NET Core 8 Web API
- Entity Framework Core
- ASP.NET Core Identity
- JWT Bearer Authentication
- SQL Server LocalDB
- Swagger UI

### Frontend
- React
- TypeScript
- Vite
- React Router
- TanStack Query
- Ant Design
- Vitest
- React Testing Library

## Основные возможности по ролям

### Client
- создать заявку
- просмотреть свои обращения
- открыть детали заявки

### Master
- видеть свободные заявки
- брать заявку в работу
- менять статус заявки
- отклонять заявку
- просматривать завершённые заявки

### Admin
- управлять категориями
- управлять пользователями и ролями
